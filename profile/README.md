# AI EMS
### 차세대 응급환자 이송 관제 시스템

</div>

<div align="center">

## 개요

</div>

AI EMS는 응급환자 이송 과정에서 병원 선정에 소요되는 시간을 단축하여 환자의 골든타임을 확보하는 차세대 응급 이송 관제 시스템입니다.

구급대원이 여러 병원에 개별적으로 전화하는 비효율적인 방식을 디지털 플랫폼 기반의 통합 관제 시스템으로 전환하여, 구급차와 복수 병원 간 실시간 동시 커뮤니케이션을 가능하게 합니다.

<div align="center">

## 특징

</div>

- **음성 인식 기반 핸즈프리 시스템**: 구급대원이 환자 처치에 집중할 수 있도록 STT 기술 활용
- **실시간 병원 매칭**: AI 기반 알고리즘으로 환자 상태에 최적화된 병원 자동 선별
- **동시다발 통신**: 복수 병원에 동시 요청 전송으로 응답 시간 단축
- **표준화된 데이터 관리**: 구조화된 환자 정보 및 이송 기록 자동화

<div align="center">

## 시스템 구성

</div>

### 1. 구급대원용 태블릿 앱 (Ambulance)
- 기기 인증 및 운행 상태 관리
- STT 기반 환자 정보 입력 및 구조화
- 실시간 GPS 위치 추적
- 병원 매칭 및 선택 기능
- 구급활동일지 자동 생성

### 2. 병원 응급실 대시보드 (ER)
- 이송 요청 실시간 수신 및 응답
- 환자 정보 상세 조회
- 구급차 위치 추적 및 ETA 확인

### 3. 중앙 관제 시스템 (CCMS)
- 전체 앰뷸런스 운행 현황 모니터링
- 병원 매칭 알고리즘 처리
- 실시간 데이터 중계 및 관리
